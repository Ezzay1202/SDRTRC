## 3. β 分析

### 3.1 β 分布

```
β = 0.000  ■■■■■■■■  (8/30, 26.7%)  → SDRTR 残差完全不可靠
β ∈ (0, 0.3]  ■■■■■  (5/30, 16.7%)  → 微量使用
β ∈ (0.3, 0.6]  ■■■■■■■■  (8/30, 26.7%)  → 中等信任
β ∈ (0.6, 0.9]  ■■■■■■■  (7/30, 23.3%)  → 较强信任
β > 0.9       ■■  (2/30,  6.7%)  → 几乎完全信任 SDRTR
```

### 3.2 各数据集 β 均值

| 数据集 | β 均值 | 含义 |
|--------|:------:|------|
| ETTh1 | 0.264 | SDRTR 信号弱 |
| ETTh2 | 0.025 | SDRTR 基本无信号 |
| ETTm1 | 0.819 | SDRTR 信号较强 |
| ETTm2 | 0.658 | SDRTR 有一定信号 |
| Weather | 0.484 | 中等 |
| Electricity | 0.424 | 中等 |
| Traffic | 0.321 | 偏弱 |
| **HogPrice** | **0.000** | **完全无信号** |

### 3.3 β 机制说明

SDRTRC 校准公式：

```
pred_SDRTRC = pred_XLinear + β × (pred_SDRTR − pred_XLinear)
```

- **β = 0**: SDRTR 的残差预测不包含任何可靠信息 → SDRTRC = XLinear
- **β = 1**: SDRTR 的残差预测完全可信 → SDRTRC = SDRTR
- **β ∈ (0, 1)**: 部分信任，按比例混合

β 通过验证集上最小化 MSE 学习得到（scalar 模式：一个 β 值应用于所有预测维度）。

## 5. 结论

### 5.1 核心发现

**SDRTR v3.1（TRPLResidualBranch + FreqStateEncoder）未能对 XLinear 基线提供实质性改善。**

1. **27% 的组合 β=0**，校准器自身判定 SDRTR 残差不含有效信号
2. **中位改善仅 0.00034 MSE**，在 random seed 噪声量级之内
3. **HogPrice 数据集**上 SDRTR 完全无效（β=0 @ 96+192），而该数据集是论文的核心应用场景
4. 即使 SDRTRC "胜出"的 16 个组合，改善幅度也不足以排除随机性

### 5.2 架构层面诊断

SDRTR 的核心假设是：channel-wise 状态依赖残差学习能捕捉 XLinear 简单线性映射之外的信号。

v1 → v2.1 → v2.2 → v3.1 四轮迭代的实验结果表明，这个假设在当前模型设计下不成立。Attention-based 残差依赖学习器引入的复杂度未能转化为预测能力的提升。

### 5.3 论文写作建议

- 诚实地报告 SDRTRC 未超越 XLinear 的事实
- β 校准机制本身可作为亮点：它诚实揭示了残差无信号，避免了过拟合
- 考虑将 XLinear + β 校准框架作为核心贡献（而非 SDRTR 架构）
